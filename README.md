# Three White Towers

This project is a hands-on learning experiment focused on modern backend, infrastructure, and API patterns. It leverages Rust, cloud-native technologies, and generative AI (Google Gemini, Vertex AI) to explore, prototype, and document best practices in software engineering.

## Purpose
- Experiment with and learn about backend and infrastructure approaches
- Integrate generative AI (Google Gemini, Vertex AI) for code, automation, and knowledge sharing
- Rapid prototyping and iterative development

## Technologies
- Rust
- Cloud-native tools
- Generative AI (Google Gemini, Vertex AI)

## Approach
Build, break, and iterate to maximize learning and share insights gained throughout the process.
