# Three White Towers - Infrastructure

This directory is part of a broader learning experiment. The project explores modern backend, infrastructure, and API patterns using Rust, cloud-native technologies, and best practices. The goal is hands-on experimentation, rapid prototyping, and sharing knowledge gained throughout the process.

- **Purpose:** Experiment, learn, and document modern backend and infrastructure approaches.
- **Tech:** Rust, cloud-native tools, modern API design.
- **Approach:** Build, break, and iterate to maximize learning.
